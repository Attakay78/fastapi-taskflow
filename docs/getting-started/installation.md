# Installation

## Requirements

- Python 3.10+

## Install

```bash
pip install fastapi-taskflow
```

## Optional: Redis backend

```bash
pip install "fastapi-taskflow[redis]"
```

## Development install

```bash
git clone https://github.com/Attakay78/fastapi-taskflow
cd fastapi-taskflow
pip install -e ".[dev]"
```
